Vestibular 2024 - Analise e Desenvolvimento de Sistemas

As 5 Principais Diciplinas Deste Curso !
  1. Arquitetura e Organização de Computadores
    • A arquitetura de computadores é a forma como os diversos componentes de um computador
      são organizados, determina aspectos relacionados à qualidade, ao desempenho e à
      aplicação para a qual o dispositivo vai ser orientado.
  2. Comunicação Empresarial
    • A Comunicação Empresarial ou Comunicação Corporativa é uma atividade estratégica para as diretorias e presidências das empresas.
  3. Programming Basics: Criação de Aplicações e Sistemas
    • A criação de aplicações e sistemas de informação é um campo de estudo que envolve o desenvolvimento de software e sistemas para coletar, processar, armazenar e disseminar dados informacionais.
  4. Análise e Levantamento de Requisitos de Software
    • A análise de requisitos de software é uma das etapas mais importantes do desenvolvimento de um projeto.
  5. Object-Oriented Development: Programação Extrema, Design de Software, Empreendedorismo e
    Criação de Novos Negócios
    • A programação orientada a objetos é um modelo de análise, projeto e programação de software baseado na composição e interação entre diversas unidades chamadas de “objetos”.
